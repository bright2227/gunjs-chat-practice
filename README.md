# gunjs-chat-practice

gunjs is decentralized db using web browser's localStorage. There is no bootstrap website, and still needs a backend as relay.

But it can act as a kind of webSockect module.

### 1. start relay

``` cd relay ```

``` npm install ```

``` node main.js ```

### 2. start client

``` cd client ```

``` npm install ```

``` npm start ```
